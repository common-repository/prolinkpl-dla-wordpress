Wyświetl sortowalną listę wszystkich reklam umieszczonych na Twoim blogu za pośrednictwem platformy reklamowej ProLink.pl. 



== Description ==

Używasz ProLink.pl? Zobacz listę wszystkich reklam wyświetlanych na Twoim blogu wprost w panelu administracjnym WordPressa. 

* Automatycznie wykrywa kod serwisu
* Wyświetla wszystkie reklamy wyświetlane w danym serwisie w formie sortowalnej tabeli
* Zlicza i wyświetla liczbę wszystkich reklam wyświetlonych w danym serwisie, liczbę unikalnych adresów docelowych oraz unikalnych reklamodawców 

Ważne: ProLink.pl stosuje w swoim pliku hashe zamiast adresów danych podstron na których są opublikowane. Czasem robot indeksuje dany link z jakimś parametrem, czasem stronicowanie - odgadnięcie wszystkich linków może być niemożliwe. W tych przypadkach w tabeli wyświetlane jest oznaczenie "[ ? ]".

Nie używasz jeszcze ProLink.pl? Zarejestruj się już teraz: http://bit.ly/prolinkpl 



EN: 

This plugin is dedicated to the Polish advertising platform ProLink.pl. It lists all the ads that are placed on the site in form of a sortable table.


== Installation ==

1. Rozpakuj archiwum i wgrał cały katalog `pldwp` do katalogu z wtyczkami (zazwyczaj `/wp-content/plugins/`)
2. Aktywuj plugin w panelu WordPress-a

== Frequently Asked Questions ==

= Czy plugin wymaga podania hasła do systemu ProLink.pl? =

Nie. Plugin w żaden sposób nie łączy się systemem ProLink.pl, tylko analizuje plik z listą reklam, który znajduje się na Twoim serwerze.
